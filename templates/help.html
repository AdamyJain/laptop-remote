Mac Setup Guide
Get Laptop Remote running on macOS in under two minutes.
01 — Getting started

Download & run the client

1
Download the client
Go to the home page and click Download for Mac. No installation required — it's a single file.
2
Run it
Double-click LaptopRemote. A setup page will open automatically in your browser showing the URL to share with your phone.
Gatekeeper warning? macOS may block the app the first time with "cannot be opened because it is from an unidentified developer." To allow it: right-click → Open → Open in the dialog that appears. You only need to do this once.
3
Copy the URL and open it on your phone
Tap Copy URL on the setup page, then open it in your phone's browser. The setup page will update automatically once your phone connects.
02 — Permissions

Grant Accessibility access

⚠️
Required on macOS. Without this, pyautogui cannot move the mouse or send keystrokes — the phone will connect but nothing will happen on screen.
1
Open System Settings
Go to System Settings → Privacy & Security → Accessibility.
2
Add LaptopRemote
Click the + button and add LaptopRemote from wherever you saved it. Toggle it on.
3
Restart the client
Quit and relaunch LaptopRemote after granting permission. macOS won't apply the change to a running process.

04 — Mac tip

Enable Mac modifiers

By default the hotkeys (Copy, Paste, Undo, etc.) use Ctrl. On a Mac you want ⌘ Command instead.

Tap the ⚙ gear icon in the top-right of the remote and toggle Mac modifiers on. The setting is saved automatically.

05 — Troubleshooting

Common issues

?
Phone connects but cursor doesn't move
Accessibility permission is missing or wasn't applied. Follow section 02 above and restart the server.
?
Can't reach the server from the phone
Both devices must be on the same Wi-Fi network. Check that your Mac firewall isn't blocking port 5000 (System Settings → Network → Firewall).
?
BAD TOKEN / OFFLINE shown on phone
The token in the URL doesn't match the one the server generated. Copy the URL fresh from the setup page at http://localhost:5000.
